# AWS Cloud Club – University of Melbourne

## Getting Started

```bash
npm install
npm run dev
```

Then open http://localhost:5173 in your browser.

## Build for Production

```bash
npm run build
npm run preview
```

## Tech Stack

- React 19 + Vite
- Tailwind CSS v4
- Framer Motion
- Spline 3D (@splinetool/react-spline)
- React Hook Form + Zod
